# "Список задач"

## Installation
```bash
npm install
```
#### or
```bash
yarn install
```
#### or
```bash
pnpm install
```
## Running the app

```bash
npm run dev
```
#### or
```bash
yarn dev
```
#### or
```bash
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
